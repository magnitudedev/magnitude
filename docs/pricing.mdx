## Cloud

Cloud models run on our infrastructure when you choose to use them. Zero data retention applies to your prompts and code. Magnitude's open-source agent remains available with local models without a cloud subscription.

### Magnitude Pro

Pro is \$10 for the first month, then \$20/month. It includes \$10 every 5 hours, \$20 each week, and \$40 each month.

Usage is measured from each completed request at the underlying model's per-token cost. It is included in the plan rather than billed as an overage. There are no credit top-ups or automatic reloads.

Manage your plan on the [billing page](https://app.magnitude.dev/billing). Run `/usage` or `/limits` in the CLI to see current limits, reset times, and recent usage.
