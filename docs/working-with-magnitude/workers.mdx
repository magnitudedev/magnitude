This page covers how delegated work appears in the interface.

## How delegated work appears

When the lead agent delegates work, you will see brief lifecycle updates in the main conversation.

Workers also appear as tasks near the composer. Use the task list to view worker status and progress.

## The task list

A task list above the composer shows:
- the **Main** chat context,
- one task row per active or idle worker.

Task rows show live status so you can monitor work without opening details.

## Viewing details and nested delegation

Open a worker's task to view that worker's messages and reasoning stream.

If that worker delegates further, you can continue drilling into nested activity from its detail view.
