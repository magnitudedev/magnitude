## Chat interface

Type a message and press **Enter** to send. The agent will stream its response in real-time.

- **Shift+Enter** — insert a newline (multiline input)
- **Escape** or **Ctrl+C** — interrupt the agent while it's streaming
- **Escape with running subagents** — press once to interrupt current activity, press again to kill all running subagents

### Think blocks

When the agent is working, you'll see collapsible think blocks showing what it's doing — file reads, edits, searches, shell commands. These collapse automatically when the agent finishes a step.

### Fork activity cards

When the orchestrator launches subagents, you'll see inline fork activity cards in the conversation. These cards show what each subagent is doing and make parallel work easy to follow in real time.

### Artifacts

Artifacts appear inline as clickable chips like `[≡ plan]` or `[≡ findings]`. Click a chip to open the artifact in the side panel, where you can read the full content and track what agents are sharing across steps.

## Slash commands

| Command | Aliases | Description |
|---------|---------|-------------|
| `/new` | — | Start a new conversation |
| `/resume` | — | Resume a previous conversation |
| `/exit` | `/quit`, `/q` | Exit Magnitude |
| `/bash` | — | Enter bash mode |
| `/init` | — | Generate AGENTS.md for this project |
| `/settings` | `/s` | Open settings (providers, models) |
| `/browser-setup` | — | Configure browser automation prerequisites |
| `/model` | `/m` | Quick model picker |
| `/provider` | — | Manage providers |

Skills you've added also appear as slash commands — type `/<skill-name>` to activate them.

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| **Enter** | Send message / approve |
| **Shift+Enter** | Insert newline |
| **Escape** | Interrupt streaming (once), kill all running subagents (twice), exit bash mode, close overlays |
| **Ctrl+C** (with text) | Clear input |
| **Ctrl+C** (2x tap) | Exit Magnitude |
| **Ctrl+R** | Open recent chats |
| **Ctrl+D** | Toggle debug panel (requires `--debug`) |
| **A** | Approve tool call |
| **D** | Deny tool call |
