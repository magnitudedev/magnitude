Bash mode lets you run shell commands directly without going through the agent.

## Entering bash mode

- Type `!` as the first character of your message
- Or run `/bash`

The border turns orange and shows a **BASH** label to indicate you're in bash mode.

## Running commands

Type any shell command and press Enter. Output (stdout, stderr, exit code) is displayed directly in the chat.

## Exiting

Press **Escape** to return to normal chat mode.
