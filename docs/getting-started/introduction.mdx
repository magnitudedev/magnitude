# What is Magnitude?

Magnitude is an open source AI coding agent that runs in your terminal.

It reads your codebase, makes edits, runs commands, and searches the web — all through a conversational interface.

<img src="/images/interface.png" alt="Magnitude Interface" />

Magnitude uses an advanced orchestrator → subagents model for better context management. The orchestrator talks to you and deploys specialized agents to research, plan, build, and verify changes. Each agent gets a focused context window, which means far less compaction, better performance on large tasks, and a superior overall experience.

## Why Magnitude?

- **Less compaction.** Focused context windows mean far less dreaded "Compacting conversation..."
- **Better performance on large tasks.** Doesn't lose the thread over complicated, long-horizon tasks.
- **Built-in browser agent.** Uses a native browser agent to verify UI changes and behavior.
- **Model agnostic.** Frontier models via API, open source models, local models, whatever you want.
