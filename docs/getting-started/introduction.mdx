# What is Magnitude?

Magnitude is an open source coding agent where everything happens in specialized subagents. The main agent stays focused on your intent, leading to stronger performance on longer tasks and higher quality code.

<img src="/images/interface.png" alt="Magnitude Interface" />

The main agent manages the conversation and delegates to specialized subagents — explorer, planner, builder, reviewer, browser, and debugger — each with its own role, specific context, and toolset. Subagents do focused work and report back, keeping the main agent's context clean.

## Why Magnitude?

- **Keeps the main agent focused.** Implementation happens in subagents, so the main agent stays focused on your intent across longer tasks.
- **Supports direct steering.** Subagents can surface blockers, ask clarifying questions, and receive direct messages when you need to redirect work.
- **Lets you mix models.** Use different providers and models for orchestration, implementation, and browser work.
- **Scopes tools by role.** Each subagent gets the tools it needs for its job instead of relying on manual mode switches.
- **Stays interactive during execution.** You can continue the main conversation while delegated work is still running.
